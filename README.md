# タスクボード

短期・中期・長期で分けて管理できるカンバン型タスク管理アプリ（React + Vite）。

## 機能
- **作成日付が自動入力** — タスク作成時に日付が自動で記録されます。
- **1クリックで分類** — 短期 / 中期 / 長期の各列で入力 → ＋ で即追加。カード下のボタンで他の分類へ1クリック移動。
- **完了の視覚化** — チェックで文字に取り消し線。「完了フォルダへ」ボタンで完了フォルダに移動。
- **追加タスク（サブタスク）** — 各タスクに「＋ 追加タスク」で子タスクを追加（1階層）。進捗（◯/◯）も表示。
- **色分けUI** — 短期=赤 / 中期=青 / 長期=緑 で直感的に区別。
- **ファイル保存** — 「書き出し」でJSONを保存、「読み込み」で復元。普段の編集はブラウザに自動保存されます。

## 起動方法
```bash
npm install      # 初回のみ
npm run dev      # 開発サーバー起動 → 表示されたURLをブラウザで開く
```

## ビルド（配布用）
```bash
npm run build    # dist/ に静的ファイルを生成
npm run preview  # ビルド結果をローカルで確認
```

## 操作メモ
- タスク名は**ダブルクリック**で編集できます。
- 「書き出し」したJSONファイルがバックアップになります。別PCへ移すときは読み込みで復元してください。
